Developer_ReadMe

Repository: Support

Description

This folder contains projects to set up our basic development environment.

  • Building project CCDev configures and sets up our basic development.
  • Projects shunit2 and ASUnit are in-house scripts to install third-party products.

Dependencies

Project CCDev should be installed first, since it performs configuration and installs scripts that are needed for other development work.

CCDev automatically builds shunit2 and ASUnit.
