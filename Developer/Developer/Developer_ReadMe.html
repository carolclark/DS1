Developer_ReadMe

Repository: Support

Description

This folder contains projects to set up our basic development environment.

  • script "Developer/DevConfig.ksh" DevConfig configures and sets up our basic development.
  • Projects shunit2 (and ASUnit) are in-house scripts to install third-party products.
  • script Developer_install.ksh runs the above scripts as indicated, and installs many other items itself.

Dependencies

script DevConfig.ksh should be performed first, since it performs configuration and installs scripts that are needed for other development work.

this target automatically builds shunit2( and ASUnit, if needed later).
