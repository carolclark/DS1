CCDev CCDev_ReadMe

Repository: Support

Description

Building project CCDev configures and sets up our basic development.

Customization for different developers is supported.

Tasks include:

  • install .profile, .kshrc, .launchd.conf files
    • set Korn shell; set up Python and shunit paths
    • define DEV=${HOME}/Dev (development work area), CCDev=${HOME}/Library/CCDev (derived content)
    • configure git, Terminal, Xcode
  • install ksh and Python scripts, Automator workflows
  • compile (in a separate target) and install AppleScripts
  • run shunit tests

Dependencies

depends on shunit::shunit
