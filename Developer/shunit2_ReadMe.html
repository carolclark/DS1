Repository: Support

Description

shunit2 is a Third-party script to test shell code. This project:

  • installs the script, along with tests and supporting files, in a folder at ${HOME}/Libraries/CCDev/shunit
  • runs shunit's included tests for each supported shell

Note that failure of the test script does NOT generate a test failure in Xcode.

Documentation and examples are also included with the source code, but are not installed.
