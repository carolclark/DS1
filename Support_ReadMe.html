Repository: Support

Description

Repository Support contains support for our development environment and accessibility needs.
This includes scripts, application support, and documentation support.

Project Support includes subprojects Docs, Developer, BuildSupport (being phased out), and ThirdParty.
ThirdParty provides build and some revision-checking for non-Cocoa tools, which cannot use CocoaPods.
