Python

Errors, Logging, and Debugging

Python Debugging

  • to debug from Terminal using Python debugger pdb: python -m pdb ${CCDev}/bin/python/<filename> <arguments>
  • Debugger Commands: see Debugger Commands
    • if debugger will not quit, type "reset", then quit

Logging

	* standard Python template imports logging module and configures logging
	* logging levels: CRITICAL, ERROR, WARNING, INFO, DEBUG
	* can also:
		* pass extra information for use with logging format; see logging.debug
		* pass exception to be added to logging message;  see logging.debug > logging.exception
			* for use only from an exception handler
