Cdoc

(info in DevSupport?) Cdoc is system, internal to C & C software, for producing accessible HTML technical documentation, with support for special features including versioning and revision history information. See the topics below for information on the available features and how to include them in documentation files.

Cdoc files require processing to translate its special features into standard HTML format. See ccInstall for information about building Cdoc documentation from source.
